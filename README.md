# STUDENT UTILITY APP
## DESCRIPTION
The student utility app is a simple CLI based application, meant to serve as practice for the learning of basic programming concepts such as variable types, operators, decision making, loops, and handling user inputs in Java.
## HOW TO RUN THE PROGRAM

1. Clone the repository:

```bash
git clone https://github.com/yourusername/java-student-utility-app.git
```

2. Navigate into the source directory:

```bash
cd java-student-utility-app/src
```

3. Compile the program:

```bash
javac StudentUtilityApp.java
```

4. Run the program:

```bash
java StudentUtilityApp
```
### Running it in IntelliJ IDEA

1. Open IntelliJ IDEA.
2. Select "Open" and choose the project folder.
3. Locate the `StudentUtilityApp` class inside the `src` directory.
4. Click the Run button or right-click the file and select "Run".
